##Ari

Ari means ant in japanese, the reason of the name is the Aesop's Fable 'The Ant and the Grasshopper'.


###Prerequisites

 * Java 8
 * Maven 3.0

####Build

mvn clean package

####Test

mvn clean test

####Run

java -jar target/kuwagara-0.1.0.jar

####Acceptance Test

* cd acceptance-test
* mvn test
